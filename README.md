# 📝 ToDo List App

A simple and efficient task management application with localStorage persistence.

![App Screenshot](public/images/1.png)

## ✨ Features

- ✅ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✔️ Mark tasks as complete/incomplete
- 💾 Automatic saving to localStorage
- 🎨 Beautiful UI with Tailwind CSS

![App Screenshot](public/images/2.png)

## 🛠️ Technologies

- **React 19** - Core UI library
- **TanStack Router** - Routing management
- **Tailwind CSS 4** - Styling
- **Phosphor Icons** - Beautiful icons
- **UUID** - Unique ID generation for tasks
